# Crypto Analytics Project

## Overview
This project fetches real-time crypto data from CoinGecko API,
processes it, stores it in SQLite, and generates insights.

## Tools Used
- Python
- SQL
- Power BI
- Excel

## Features
- Data pipeline automation
- SQL analysis
- Dashboard visualization
- Business insights

## Sample Insights
- BTC dominance increased by X%
- Top 5 coins drive majority market cap
